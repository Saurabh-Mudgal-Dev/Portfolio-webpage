# Project Overview

## Purpose of Project
The primary objective of this project was to create a digital identity where my academic capabilities can be shown. It moves beyond simple file storage, presenting my coding projects in a visually engaging, professional format suitable for sharing them with potential recruiters or peers.

## Scope of Project
The portfolio currently features:
1.  **Personal Branding:** A "Hero" section with a brief introduction and profile visual.
2.  **Projects:** A few projects that i have worked on.
3.  **Contact Hub:** Direct links to professional networks (GitHub, LinkedIn) and email contact.
4.  **WebDev capability** It shows how i can make entire websites which is a skill in itself.

## Tech Stack
* **HTML5:** Used for semantic structuring (header, section, footer).
* **CSS3:** Used for layout (Flexbox), responsiveness (Media Queries), and visual flair (Gradients, Animations).
* **Git/GitHub:** Used for version control and hosting.
